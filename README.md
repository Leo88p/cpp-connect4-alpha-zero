# AlphaZero implementation for Connect 4
This is an implementation of AlphaZero for Connect 4 on C++. Connect 4 game model is based on [Pascal Pons'](https://github.com/PascalPons/connect4). Training process was optimized for GTX 1650. Project also includes python bindings for convinient testing of models using python.
## Dependencies
* libtorch 2.9.1+cu130
* NVIDIA CUDA Toolkit 13.1
* Python 3.11
